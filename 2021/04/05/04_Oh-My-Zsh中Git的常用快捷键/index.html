Oh-My-Zsh中Git的常用快捷键

本文最后更新于：2021年8月8日 早上

每次使用Git都要输入太多重复内容？Oh-My-Zsh的Git捷键帮到你~🤩

快捷键

下面将展示 Oh-My-Zsh 中 Git 的快捷键，第一部分的“常用快捷键”已经涵盖了日常大部分使用场景。

常用快捷键

快捷键 完整写法
g git
gaa git add .
gcmsg git commit -m
gcb git checkout -b
gst git status
gf git fetch
gl git pull
gup git pull –rebase
grb git rebase
gsta git stash save
gstp git stash pop
gcp git cherry-pick
mg git merge
gp git push
